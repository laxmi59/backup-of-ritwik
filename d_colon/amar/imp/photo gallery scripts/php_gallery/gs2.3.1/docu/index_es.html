Configuracion

Despues de desempaquetasr el sript debe ser capaz de ejecutarlo directamente.

Si quiere utilizar una carpeta de imagenes diferente a la que viene por defecto puede cambiar la ruta en la variable $image_url. Introduzca la ruta completa en el servidor (p.ej. /usr/local/etc/htdocs/www/images/picture_gallery/).

La variable $image_url es la direccion de las imagenes para llamarlas desde las paginas HTML. Puede utilizar la direccion relativa (valor por defecto) o la direccion web completa a la carpeta de las imagenes (p.ej.: si la direccion completa a la imagen es http://www.server.com/images/picture.gif, la ruta absoluta sera '/images/picture.gif').

La variable $order contiene el metodo de ordenacion. Estan disponibles 'ascendente' y 'descendente'. El orden se basa en los nombres de archivo. Finalmente necesita configurar el idioma. Estan disponibles Ingles, Aleman y Espanol. El idioma por defecto es el Espaool.

Personalizar la presentacion

El directorio 'templates' contiene los archivos para la presentacion. Puede utilizar HTML y CSS para personalizar la presentacion.

Cada una de las plantillas 'index.html' y 'window.html' contiene el esquema de la pagina. Las plantillas 'table.html' y 'window_table.html' contiene el esquema de las filas de las tablas. El script toma cada celda y la transforma en una tabla completa.

En caso de utilizar un editor WYSIWYG como Dreamweaver, Frontpage o NetObjects, por favor, asegœrese de que el software no hace cambios por s' mismo.

Los archivos de plantilla contienen marcadores con { y } como: {txt_next_page}. Estos marcadores seran reemplazados automaticamentepor sus valores correspondientes desde el archivo de idioma 'languages/language.es.inc.php'. Usted es libre de editar ese archivo y cambiar las palabras y frases. Tambien puede reemplazar los marcadores dentro de las plantillas por palabras reales o poner esos marcadores en otras posiciones dentro de la plantilla. Y tambien puede mejorar el archivo de idioma utilizando el mismo patron.

Entre estos marcadores existen los llamados Loops. Similares a las etiquetas HTML tags las dos partes (...) no deben separarse. Si quiere alterar la posicion en la plantilla, debe mover todo el bloque.

Las Imagenes

Las imagenes se ordenaran tal y como esta definido en la variable $order. Con el nombre de la imagen se puede influir en la posicion de esta.

Tambien existe la posibilidad de poner un archivo 'image_order.txt' en su directorio de imagenes. El archivo contiene los nombres de imagen en el orden que se quiera (uno debajo de otro).

La carpeta de imagenes de la version 'simple' *NO* contiene un 'image_order.txt'. Las otras dos versiones s' lo contienen.

Si esta trabajando con miniaturas y las correspondientes imagenes grandes, los nombres de las imagenes deben ser los mismos.

Por favor, asegœrese de que su directorio de imagenes contiene solo los archivos que quiere visualizar con el script. Si no quiere utilizar el 'image_order.txt' debe eliminarlo. Renombrando 'image_order.txt' no funcionara y causara errores.

Subir los archivos de script

Cree las siguientes carpetas en su servidor web:

/gallery/
/gallery /inc/
/gallery /templates/
/gallery /images/

Copie todos los ficheros en sus correspondientes carpetas.

Ahora el script debiera funcionar.

Por favor, tome nota: Usted puede ejecutar varias galer'as con las mismas plantillas de esquema. Copie los ficheros 'index.php' y 'window.php' dandoles un nuevo nombre. Ahora solo necesita cambiar la ruta al directorio de imagenes.
