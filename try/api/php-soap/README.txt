4PSA VoipNow SOAP API Client for PHP
version 0.0.1
===================================

Disclaimer
----------

This demonstrative code SHOULD NOT be used in production. It is designed to show how a client application can interact with the 4PSA VoipNow SOAP API. Therefore there are no validations and no error checking is relaxed, in order to demonstrate the most common mistakes.


Requirements
------------

PHP 5.1.2 or later is REQUIRED by this software.


Installation
------------


1. Copy all files to a directory that can be accessed over the web

2. Open config.php with an editor and adjust the variables according to your setup. Pay attention to IDs.

3. Access the file index.php over the web with a browser.
